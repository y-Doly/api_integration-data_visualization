# COVID-19 Data Visualization Project

##  Project Overview
This project fetches real-time COVID-19 historical data using a public API and visualizes the data using Python's Matplotlib library.

The goal of this project is to demonstrate:
- API Integration
- Data Processing
- Data Visualization
- Basic Data Analysis

---

##  Features
- Fetches COVID-19 data for India (last 30 days)
- Plots total cumulative cases
- Displays daily new cases
- Clean and readable graphical dashboard

---

##  Technologies Used
- Python
- Requests (API handling)
- Matplotlib (Data Visualization)

---

##  API Used
Data Source:
https://disease.sh/

Endpoint:
https://disease.sh/v3/covid-19/historical/India?lastdays=30

This API provides historical COVID-19 data for countries worldwide.
